# 微信开发验收版说明

## 当前配置

- 目标 AppID：`wxf16ad96ba7ac7238`
- 接口基址：`http://8.136.34.168`
- 登录策略：后端开启 `mock-user-login`
- 支付策略：后端开启 `mock-payment`，前端命中 `mockPay` 分支

## 导入方式

1. 使用 HBuilderX 编译到微信小程序，或直接导入 `unpackage/dist/dev/mp-weixin`
2. 在微信开发者工具中确认项目 AppID 为 `wxf16ad96ba7ac7238`
3. 保持开发配置 `urlCheck=false`

## 验收链路

1. 启动小程序
2. 完成微信登录
3. 浏览商户、分类和菜品
4. 加入购物车并提交订单
5. 进入支付页并完成 Mock 支付
6. 查看成功页、订单详情和历史订单

## 使用边界

- 当前方案只用于微信开发者工具中的功能验收
- 当前方案不是微信官方体验成员版
- 当前方案不能直接作为正式版提交审核
- 如果后续要做真正的体验版或正式版，必须补齐域名、HTTPS 和微信后台合法域名配置
