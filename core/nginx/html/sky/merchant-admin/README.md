# 校园外卖管理系统

一个现代化、视觉精美的外卖管理后台系统，专为餐饮商家设计。

## ✨ 特性

### 🎨 设计亮点
- **现代化UI设计**：采用温暖的橙色系配色，符合外卖品牌调性
- **响应式布局**：完美适配桌面端和平板设备
- **流畅动画**：卡片悬停、页面切换等细腻交互动效
- **数据可视化**：集成 Chart.js 图表库，直观展示经营数据

### 📱 核心功能模块

#### 1. 仪表盘 (Dashboard)
- 实时统计卡片：今日订单、营业额、待处理订单、商品总数
- 营业趋势图：7天销售数据折线图
- 分类销售占比：饼图展示各品类销售情况
- 最新订单流：实时显示最新 5 条订单

#### 2. 商品管理 (Products)
- 卡片式商品展示（支持 4 列网格布局）
- 多维度筛选：商品名称搜索、分类筛选、状态筛选
- 商品操作：上下架切换、编辑、删除
- 商品信息：图片、名称、价格、分类、销量、库存

#### 3. 订单管理 (Orders)
- 状态标签页切换：全部、待接单、备餐中、配送中、已完成
- 订单卡片详情：订单号、顾客信息、商品清单、总金额
- 订单操作流：接单 → 开始配送 → 完成订单
- 实时订单提醒（顶部通知栏红点提示）

#### 4. 套餐管理 (Set Meals)
- 套餐卡片展示：主图、名称、价格、优惠标签
- 套餐详情：包含商品列表、原价对比、销量统计

#### 5. 数据统计 (Statistics)
- 双轴图表：订单量与销售额趋势对比
- 时段分析：24小时订单量柱状图（热力色标）
- 热销商品 Top 10：排行榜 + 进度条可视化
- 分类销售详情：各品类销售占比统计

#### 6. 员工管理 (Employees)
- 员工列表表格：姓名、手机号、职位、状态、入职时间
- 员工状态管理：在职/离职标签

### 🛠 技术栈

- **前端框架**：原生 JavaScript (ES6+)
- **UI样式**：Tailwind CSS (CDN)
- **数据可视化**：Chart.js
- **图标系统**：Heroicons (SVG)
- **架构模式**：SPA (Single Page Application)
