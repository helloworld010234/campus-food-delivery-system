## 🔐 登录测试

1. 打开前端页面
2. 输入账号密码：
   - **用户名**：`admin`
   - **密码**：`123456`
3. 点击登录

### 可能的问题及解决

#### 问题1：登录失败 "用户名或密码错误"
**原因**：数据库中没有该账号或密码不匹配

**解决**：
```sql
-- 查询现有账号
SELECT username, password FROM employee;

-- 如果需要重置密码为 123456
UPDATE employee SET password = 'e10adc3949ba59abbe56e057f20f883e' WHERE username = 'admin';
```

#### 问题2：CORS 跨域错误
**症状**：浏览器控制台显示 `Access-Control-Allow-Origin` 错误

**解决**：
1. 确认后端 `WebMvcConfiguration.java` 已添加跨域配置
2. 重启后端服务
3. 清除浏览器缓存

#### 问题3：连接被拒绝
**症状**：`Failed to fetch` 或 `Connection refused`

**解决**：
1. 确认后端服务已启动（访问 http://localhost:8080/doc.html）
2. 检查前端 API 配置的地址是否正确：
   ```javascript
   // api-service.js 第 5 行
   baseURL: 'http://localhost:8080'
   ```

#### 问题4：Redis 连接失败
**症状**：后端启动时报 Redis 连接错误

**解决**：
- 启动本地 Redis 服务，或
- 修改配置文件中的 Redis 地址

## 🔧 前端 API 配置

### 修改 API 基础地址
文件：`reggie-takeout-admin/assets/js/api-service.js`

```javascript
const API_CONFIG = {
    baseURL: 'http://localhost:8080',  // 修改为你的后端地址
    timeout: 10000,
    tokenKey: 'token',
};
```

### API 使用示例

```javascript
// 登录
const result = await API.Employee.login('admin', '123456');

// 获取菜品列表
const dishes = await API.Dish.getDishList({
    page: 1,
    pageSize: 10,
    name: '宫保鸡丁'
});

// 起售菜品
await API.Dish.updateStatus(1, dishId);
```
