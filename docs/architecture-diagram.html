Sky Delivery 校园外卖系统架构

Spring Boot + UniApp 微信小程序全栈架构

前端层 — 微信小程序

  • • 框架: UniApp (Vue2) 编译至微信小程序
  • • 状态管理: Vuex + session 持久化
  • • 核心页面: 首页/商家切换/下单/支付/地址/订单/我的
  • • 网络: 封装 uni.request，JWT 鉴权，OSS 图片代理

后端层 — Spring Boot 多模块

  • • sky-server: 主服务 (Boot 3.4 + Java 17 + Tomcat)
  • • sky-common: 工具类、常量、全局异常、AOP
  • • sky-pojo: Entity / DTO / VO 数据对象
  • • 能力: MyBatis-Plus、WebSocket 推送、定时任务、JWT 鉴权

数据与外部服务层

  • • MySQL 8.0: 业务数据 (订单/菜品/用户/地址/员工)
  • • Redis: 缓存、分布式 Session、Token 黑名单
  • • 阿里云 OSS: 菜品图片/静态资源存储
  • • 微信支付: 统一下单、订单查询、退款接口
